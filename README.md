App Calcula Salário
---
Aplicativo Android desenvolvido em Java no Android Studio com o objetivo de calcular o novo salário de um funcionário com base em um percentual de aumento escolhido pelo usuário.

Funcionalidades
---
Digitar o valor do salário;

Escolher o percentual de aumento:

40%

45%

50%

Exibir o novo salário reajustado em uma caixa de diálogo.

Componentes Utilizados
---
TextView

EditText

RadioButton

RadioGroup

Button

AlertDialog

Estrutura do Projeto
---
O aplicativo possui:

Interface criada em XML (activity_main.xml);

Lógica desenvolvida em Java (MainActivity.java).

Funcionamento
---
O usuário informa o salário;

Seleciona o percentual desejado;

Clica no botão Calcular Novo Salário;

O sistema calcula e mostra o valor atualizado.

Exemplo de Cálculo
---
Salário: R$ 2000,00

Aumento: 40%

Novo salário: R$ 2800,00

Tecnologias Utilizadas
---
Java

Android Studio

XML

ConstraintLayout

Projeto desenvolvido para prática de desenvolvimento Android e manipulação de componentes visuais.
